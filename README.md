Last Uploaded Image as Featured
A WordPress plugin that dynamically sets the featured image of selected pages and posts, without manual configuration.
For each enabled page or post, you can choose between two modes:

Content mode — uses the most recently uploaded image found in the post body
Archive mode — uses the featured image of the most recently published post, with optional category include/exclude filters

The og:image meta tag is also overridden automatically, ensuring correct previews on social media and messaging apps. Compatible with Yoast SEO and any other SEO plugin.
Features

Per-page/post configuration directly from WordPress settings
Archive mode with category include/exclude filters
Automatic og:image override (Yoast SEO + generic fallback)
Optional fallback image when no image is found
No caching — always reflects the latest content in real time
